

### Project Description

**GitHub User Fetcher** is a simple web application that allows users to fetch and display information about GitHub users by entering their usernames. The application leverages the GitHub API to retrieve user details such as name, ID, bio, location, followers, and avatar image. 

### Features
- Input field for entering a GitHub username.
- Fetches user data from the GitHub API.
- Displays relevant user information, including:
  - Name
  - User ID
  - Number of followers
  - Bio
  - Location
  - Blog link (if available)
  - Avatar image
- User-friendly interface with basic styling.

### Prerequisites
- A modern web browser (e.g., Google Chrome, Firefox, Safari).

### Instructions to Run
1. **Clone or Download the Repository**: 
   - You can either clone this repository using Git or download the ZIP file.

   ```bash
   git clone <repository-url>
   ```

2. **Open the HTML File**: 
   - Navigate to the directory where you saved the project.
   - Open `github_user_fetcher.html` in your web browser.

3. **Using the Application**: 
   - Enter a valid GitHub username in the input field.
   - Click the "Fetch User Data" button to retrieve and display the user's information.

### Example Usage
1. Enter a username like `hiteshchoudhary`.
2. Click the "Fetch User Data" button.
3. View the fetched information displayed on the page.

### Troubleshooting
- Ensure you have a stable internet connection since the application makes requests to the GitHub API.
- If a user is not found or an error occurs, the application will display a message indicating the issue.
